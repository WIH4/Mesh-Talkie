# Contributing

This project is in early development. Contributions are welcome in any of the following areas:

## Areas where help is useful

- **Firmware (ESP-IDF / C++)** — Codec2 port, SX1262 driver, mesh routing, AES-256, FreeRTOS audio pipeline
- **Hardware (EasyEDA PRO / KiCad)** — schematic review, PCB layout, RF section
- **Regulatory** — SRD regulations in countries outside the EU
- **Testing** — range tests, duty cycle measurements, audio quality evaluation
- **Documentation** — protocol spec, build guides, translations

## How to contribute

1. Open an [Issue](../../issues) to describe what you want to work on
2. Fork the repo, create a branch (`feature/your-topic`)
3. Submit a Pull Request with a clear description of the change and why

## Code style

- Firmware: follow ESP-IDF conventions, C99/C++17
- Hardware: follow EasyEDA PRO naming conventions, include BOM updates

## Questions

Open an Issue or start a Discussion. No question is too basic.
